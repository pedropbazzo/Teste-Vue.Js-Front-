# VueFeedbackReaction

> Helps you to collect feedback and leads using the most spoken language in the world: the emoji.

## Installation

``` bash
# Install with npm
$ npm i 

# Running
$ npm run serve
```


## Usage

``` vue
<template>
  <div class="app">
    <vue-feedback-reaction v-model="feedback"/>
  </div>
</template>

<script>
  import VueFeedbackReaction from 'vue-feedback-reaction';
  export default {
    name: 'demo',
    components: {
      VueFeedbackReaction
    }
  };
</script>
```
